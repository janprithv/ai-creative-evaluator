# ai-creative-evaluator
An AI-powered creative evaluation platform that reviews video ads and determines launch readiness using multimodal analysis, specialized review agents, and a proprietary scoring framework.

Project Overview

AdReady AI helps brands separate polished creative from AI slop.

As AI-generated video ads become increasingly common, marketers need a scalable way to review generated ads for false claims, product inaccuracies, storyline issues, and creative brief adherence before publishing.

This project explores how multimodal AI systems can evaluate AI-generated video advertisements and provide structured feedback, rankings, and launch-readiness recommendations.

Current Scope (Subject to Change) -

Initial MVP focus:
Consumer Packaged Goods (CPG) video advertisements
AI-generated short-form video creative
Claim validation
Storyline evaluation
Product representation checks
Creative brief alignment
Launch-readiness scoring

Techniques Being Explored: 
Video transcription
OCR
Multimodal reasoning
Retrieval-Augmented Generation (RAG)
Multi-agent evaluation workflows
AI scorecarding and benchmarking

Status:
Early bootcamp project — architecture, scope, and implementation details are actively evolving.
